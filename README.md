# Book-Tracker
## Описание программы

"Book Tracker" – это приложение с графическим интерфейсом, которое поможет вам вести учет прочитанных книг. Вы можете легко добавлять новые книги, указывая название, автора, жанр и количество страниц. Приложение поддерживает фильтрацию списка книг по жанру и количеству страниц, а также автоматически сохраняет ваши данные в формате JSON, чтобы ничего не потерялось между сеансами использования.

## Инструкция по запуску

### Требования:

*   Python 3.x
*   Библиотеки: `tkinter`, `json`, `os` (входят в стандартную поставку Python)

### Запуск:

1.  Сохраните исходный код приложения (например, как `book_tracker_app.py`).
2.  Откройте терминал или командную строку.
3.  Перейдите в директорию, где вы сохранили файл.
4.  Запустите приложение командой:
    ```bash
    python book_tracker_app.py
    ```
5.  Данные будут автоматически загружаться из файла `books.json` (если он существует) и сохраняться в него при закрытии приложения или через меню "Файл" -> "Сохранить".

## Примеры использования

*   **Добавление книги:** Введите данные в соответствующие поля (Название, Автор, Жанр, Страниц) и нажмите "Добавить книгу".
*   **Фильтрация:** В разделе "Фильтрация" введите жанр или минимальное количество страниц, затем нажмите "Применить". Для сброса фильтров нажмите "Сбросить".
*   **Сохранение/Загрузка:** Используйте меню "Файл" для ручного сохранения или загрузки списка книг.
